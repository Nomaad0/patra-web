Questions fréquentes

C'est vraiment gratuit, pour toujours ?
Oui. PaTra est open source et ne nécessite aucun abonnement. Les fonctionnalités de base resteront gratuites indéfiniment. Des features avancées (multi-portefeuilles, import CSV broker) pourraient devenir payantes un jour, mais tout ce qui existe aujourd'hui reste gratuit.
Mes données sont-elles en sécurité ?
Tes données ne quittent jamais ton navigateur — elles sont stockées dans le localStorage de ton appareil. PaTra n'a pas de serveur, pas de base de données. Pour les sauvegarder, utilise le backup JSON depuis les paramètres.
Fonctionne-t-il sur mobile ?
Oui, PaTra est responsive et fonctionne sur mobile. Une application iPhone est en cours de réflexion pour une expérience encore plus fluide.
Comment les cours sont-ils mis à jour ?
Les actions (PEA/CTO) sont synchronisées via Yahoo Finance, les cryptos via CoinGecko. Tu cliques sur "Sync" et les cours se mettent à jour en quelques secondes. Aucune clé API requise.
